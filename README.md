# Ember Forecast.io

Ember addon for requesting weather forecasts from [forecast.io](http://forecast.io).


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.4 or above
* Ember CLI v2.13 or above
* Node.js v8 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-forecast-io
```
